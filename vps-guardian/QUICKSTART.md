# 🚀 Quick Start - 5 Minutos

## 1️⃣ Instalação Rápida

```bash
git clone https://github.com/seu-usuario/vps-guardian.git
cd vps-guardian
chmod +x install.sh
./install.sh
```

## 2️⃣ Acesso

- **Dashboard**: http://seu-servidor:8080
- **API**: http://seu-servidor:3000/api

## 3️⃣ Primeiros Passos

1. Acesse o dashboard
2. Explore os containers
3. Crie seu primeiro backup
4. Configure webhooks
