## 📝 Passo a Passo para GitHub

### 1️⃣ Fazer Upload

```bash
# Abra o terminal e vá para o diretório
cd /home/claude/vps-guardian

# Inicializar Git
git init

# Adicionar todos os arquivos
git add .

# Primeiro commit
git commit -m "feat: initial commit - VPS Guardian v1.0.0"

# Criar branch main
git branch -M main
```

### 2️⃣ Criar Repositório no GitHub

1. Acesse: **https://github.com/new**
2. Preencha:
   - **Repository name**: `vps-guardian`
   - **Description**: Sistema profissional de monitoramento para VPS
   - **Public** ou **Private**: Você escolhe
3. **NÃO MARQUE**: "Initialize this repository with a README, .gitignore, or LICENSE"
4. Clique: **"Create repository"**

### 3️⃣ Fazer Push

```bash
# Colar o comando que o GitHub vai mostrar
# Será algo como:

git remote add origin https://github.com/SEU-USUARIO/vps-guardian.git
git push -u origin main
```
