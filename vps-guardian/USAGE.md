# 📖 Guia de Uso

## Dashboard

O dashboard mostra:
- Estatísticas de containers
- Status do sistema
- Backups recentes
- Alertas ativos

## Containers

- **Visualizar**: Veja todos os containers com métricas
- **Controlar**: Start/Stop/Restart com 1 clique
- **Logs**: Visualize logs em tempo real

## Backups

- **Criar**: Crie backup manual
- **Restaurar**: Restaure de um backup anterior
- **Automático**: Automático diariamente às 01:00

## Configurações

Configure:
- Webhooks (Slack/Discord/Telegram)
- Backups (retenção e horário)
- Thresholds (CPU/RAM/Disco)
- Health checks
